This is a cleaned Next.js project for Vercel.

Important:
- This project is NOT Vite.
- It does NOT need src/main.jsx.
- The app entry is in src/app/

Required root files/folders:
- package.json
- package-lock.json
- next.config.ts
- next-env.d.ts
- tsconfig.json
- postcss.config.mjs
- tailwind.config.ts
- components.json
- src/

In Vercel use:
- Framework Preset: Next.js
- Build Command: npm run build
- Output Directory: leave blank

If uploading on GitHub from mobile, keep the folder paths exactly the same.
